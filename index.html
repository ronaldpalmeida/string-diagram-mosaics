String Diagram Mosaics

String Diagram Mosaics is an app for interacting with string diagrams. It is inspired by a paper of Lomonaco and Kauffman. In it they define a mosaic tiling system for studying quantum knots. This consists of a set of tiles and certain rewrite rules, which are the analogs of Reidemeister moves from knot theory. It was conjectured, and latter proven by Kuriya that this system fully captures tame knot theory. Here we define system in the same spirit, which is meant to be a presentation of a monoidal category of some sort.

How to use

Click and drag tiles from top. Click triangles to add/remove rows and columns. Click to apply a rule. A white rectangle indicates a selection on which the rule is applied. It also depends on which quadrant of the tile the mouse is on. For more, see description below.

Theory

Just definitions, no results yet. Our tiling system consists of:

  • a set $T$ of 11 square tiles, labeled $t_0$ to $t_{10}$ as shown below,
  • a set $C=\{0,1,2\}$ which encodes the type of connection a tile may have on a boundary,
  • and four "airity" maps $\alpha_i:T \to C$, where $i \in \{u,d,l,r\}$, respectively denoting the connection type of the up, down, left and right boundaries of a tile.
    • Here, $0 \in C$ means no connection point on a boundary, and $1$ and $2$ represent thin and thick lines respectively. These are meant to represent respectively the objects and morphisms of a monoidal category.

    A $(n,m)-$mosaic $M$ is an $n \times m$ matrix $M$ of tiles, subject to the following conditions.

  • It is well connected, i.e. for all $i,j$, $$ \alpha_u(M_{i,j}) = \alpha_d(M_{i-1,j})$$ $$ \alpha_l(M_{i,j}) = \alpha_r(M_{i,j-1})$$.
  • It has no points on left and right boundaries. $$\alpha_l(M_{i,1}) = \alpha_r(M_{i,n}) = 0$$
  • It is progressive, i.e. if $M_{i,j}=t_5$ and $M_{i,j+k}=t_6$ for some $k$, then there exists $l$ between $j$ and $j+k$ such that $M_{i,j}=t_3$, and likewise if $M_{i,j}=t_4$ and $M_{i,j+k}=t_3$ for some $k$, then there exists $l$ between $j$ and $j+k$ such that $M_{i,j}=t_6$.
    • The last condition ensures that there are no cups and caps. We will only focus on bare monoidal categories. Let $\mathbb{M}_{n,m}$ be the set of all $(n,m)-$mosaics.
